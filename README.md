

# 🧮 Math Toolkit

Math Toolkit is a web-based application that provides multiple mathematical tools in one platform.  
It is designed to help users perform different calculations easily through an interactive and user-friendly interface.

## 🚀 Features

The toolkit includes the following tools:

- Basic Calculator
- Scientific Calculator
- Graph Plotter
- Matrix Calculator
- Equation Solver
- Unit Converter

## 📱 Responsive Design

The website is fully responsive and works across:
- Desktop
- Tablet
- Mobile devices

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 🎯 Key Functionalities

- Keyboard and button input support
- Scientific functions and trigonometric calculations
- Graph plotting for mathematical expressions
- Matrix operations (addition, multiplication, determinant, etc.)
- Linear and quadratic equation solving
- Unit conversions for multiple categories
